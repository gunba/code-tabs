## Header Activity Visualizer
Ambient soundwave-and-mascots scene that fills the gap between the tab strip and the right-side action buttons

- [HA-01 L12] HeaderActivityViz paints the slack between .tab-bar-scroll and the right-hand action buttons (.tab-resume / .tab-config / .tab-add) into a single <canvas> as a pixel-art / 'voxel' coastal scene — sky, sun/moon, weather, layered ocean with foam, textured beach with decorations, and animated agent mascots. Backing store: dpr*cssSize with ctx.scale(dpr,dpr) and imageSmoothingEnabled=false for HiDPI crispness; ResizeObserver on the wrapper triggers needsResize and MutationObserver on document.documentElement style-attribute bumps themeBumpRef so theme/css-var changes invalidate cached tiles. All paint runs in one rAF tick — pre-rendered atlases (mascot sprites, subagent icons, sand tile, decoration tile) keep per-frame cost cheap. Layout via computeLayout(w,h): beachW = clamp(round(w*0.13), 96, 180); seaMeanY = round(h*0.6); beachShoreSlope = max(9, round(h*0.18)); waveAmpMaxPx = max(5, round(h*0.13)); beachTopYAtZero = seaMeanY - beachShoreSlope; skyHorizonY = round(h*0.45). Beach surface via shoreYAt(layout, t) with t in 0..1 — quick rise from elev=0.65 inland edge to elev=0.95 plateau (t=0.15-0.45), gentle smoothstep descent to elev=0.4 at t=0.85, sharp bank drop to elev=0 (sea level) at t=1.0; surfaceY = seaMeanY - elev*beachShoreSlope. Wave crests via computeWaveCrests on a shared Float32Array — two sin waves (period 110/47, speed 36/22), amplitude scaled by intensity (0.45..1) and ramped from 0 at x=beachW over rampDist = max(20, round(waveAmpMaxPx*3)) so sea joins beach without a step. Render pipeline: (1) drawSky — gradient by scene + horizon glow + day/night tint via celestialPhase(sunriseHour, sunsetHour); (2) drawStars at night when not fog/storm and nightAlpha > 0.15; (3) celestial body — phase.isNight ? drawMoon(cx=w-36, cy=22, r=11) : drawSun(cx=w-32, cy=18, r=8) drawn unconditionally so the sky is never empty; (4) drawClouds when scene is clouds/rain/storm (after celestial so clouds layer on top); (5) drawFog when scene==='fog'; (6) drawSeagulls on the daylight side, skipped in storm/snow/fog; (7) drawSea — gradient sea + bright crest band + sparkles (clear/clouds) + foam line along the wave crest from x=beachW to w; (8) drawMoonReflection + drawNightSeaSparkles when night and scene is clear/clouds/snow; (9) drawBeach — clipped tiled sand + decoration tile (clipped to the dune curve), then a second clipped pass for the underwater bank extension (smoothstep curve descending up to max(8, round(beachShoreSlope*0.7)) px past beachW, painted with a semi-transparent wet-sand vertical gradient rgba(110,80,40,0.55) → rgba(70,50,25,0.25) so sea bleeds through and the dune rounds off into the water), then drawBeachLandmarks (drawUmbrella at t=0.30, drawFoldoutChair at t=0.66 with back leg at t-0.04 and front leg at t+0.04 anchored to shoreYAt so the seat tilts with the local slope); (10) drawShoreChop — three+one-row foam band along the bank/water boundary, loop from round(beachW*0.45) to beachW (past the dune-crest plateau) gated by bankY >= seaMeanY - waveAmpMaxPx - 2 so foam only paints where waves can reach; bursting whitecaps anchored at xPlume = round(beachW*0.75 + ...); (11) drawSlots — agent mascots tracked in slotsMapRef Map<id, Slot> with xPx, dir, homeXPx/homeRow01, speedPxPerS, jitterSeed, diveT, bubbleAccumMs; sprites are claude-mascot.png / codex-mascot.png pre-rasterized at MASCOT_PX=22 (subagents SUBAGENT_PX=14) plus AgentTypeIcon SVG rasterized via svgToImage; active agents bounce between layout.beachW+4 and w-MASCOT_PX, idle agents rest on the beach at hash-determined homeXPx/homeRow01 (BEACH_PAD_PX=4), errored agents dive (diveT 0..1) and emit bubbles; (12) bubble trails — BUBBLE_INTERVAL_MS=320ms cadence, BUBBLE_LIFE_MS=1200ms lifetime, drifting up at 22 px/s; (13) foreground rain/snow particles via dropsRef/flakesRef; (14) storm bolt + flash every STORM_PERIOD_S=7s (STORM_FLASH_S=0.12s, STORM_BOLT_S=0.18s). makeBeachDecoTile scatters pebbles/shells/conches/starfish/driftwood and a 5-step footprint trail; both clamped to t<=0.82 (dryW = max(8, round(beachW*0.82))) so nothing drifts under the wet bank. Activity intensity samples session.metadata.toolCount deltas every 500 ms via useSessionStore.getState() (no subscription, no per-frame re-render); burst = 1-exp(-delta/3); EMA intensityRef.current * INTENSITY_DECAY_RETAIN_PER_500MS=0.7 + (BASE_INTENSITY=0.18 + 0.85*burst)*0.3 drives wave amplitude scaling; lastCounts seeds new sessions so historical activity isn't replayed as a burst. Component is aria-hidden, pointer-events:none — pure decoration. CSS shrinks to ~20 lines: .header-activity-viz (flex:1 1 0; min-width:0; overflow:hidden; pointer-events:none; border-left/right:1px solid var(--border-subtle)) wrapping a .header-activity-viz-canvas (display:block; width:100%; height:100%). Pure exported helpers (hash32, makeSlotInit, withAlpha, clampPx) plus celestialPhase and shoreYAt are unit-tested via __tests__/HeaderActivityViz.test.ts (12 cases over hash32/makeSlotInit/withAlpha/clampPx).
